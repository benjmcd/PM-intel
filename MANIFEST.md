# Manifest

This file maps source-of-truth areas for agents. Do not mirror every file; update only when ownership, generated outputs, or protected areas change.

## Command surface
| Path | Purpose |
|---|---|
| `python scripts\verify.py` | Canonical local verification command |
| `python scripts\db_local.py ...` | Local Docker Desktop/Postgres helper |
| `pmfi.cmd` | Command Prompt wrapper around `scripts\task.py` |
| `pmfi.ps1` | PowerShell wrapper around `scripts\task.py` |

## Generated or local-only paths
| Path | Rule |
|---|---|
| `.venv/` | Local virtual environment; do not commit |
| `.env` | Local secrets/config; do not commit |
| `reports/2026-06-06-alignment-audit.md` | Latest point-in-time alignment audit for this prepared workspace |
| `reports/*` | Generated runtime/backtest reports unless explicitly retained |
| `__pycache__/`, `.pytest_cache/` | Generated cache; do not commit |

## Protected areas
- `.env*` except `.env.example`.
- private keys or API credentials.
- local Postgres volumes/dumps unless explicitly requested.
- migration history after M1 unless a plan and DB verification accompany the change.
- generated files unless regenerated by a documented command.

## Removed/forbidden workflow assumptions
- No non-Windows command-runner dependency.
- No Unix-only scripts.
- No automatic agent-side command-trigger automation.
- No remote workflow or hosted deployment requirement.
- No SaaS/billing/user-account/registry-publishing preparation.
