# Agent handoff template

Append this shape to `WORKLOG.md` at meaningful boundaries.

```md
## <YYYY-MM-DD HH:MM> — <agent/tool> — <milestone>

### Goal
<What was attempted.>

### Files changed
- `<path>` — <why>

### Checks run
- `<command>` — <pass/fail + relevant output>

### Current status
<What works now.>

### Blockers / ambiguities
- <blocker or none>

### Residual risk
- <risk or none>

### Next slice
<Concrete next action for a fresh agent.>
```
