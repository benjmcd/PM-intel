# who-built-this-before-me

This repo already includes the `who-built-this-before-me` skill as a local agent workflow.

Use it when a task proposes building, shipping, forking, or materially expanding a product/tool/library. The workflow should:

1. Restate the idea.
2. Search with varied vocabulary.
3. Cluster direct, adjacent, partial, and stale prior art.
4. Extract standard patterns.
5. Benchmark differentiation.
6. Return a build/fork/contribute/use/investigate verdict.

For this repository, the skill is intended as a recurring planning discipline, not a runtime dependency.
